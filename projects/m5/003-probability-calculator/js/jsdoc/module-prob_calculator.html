(inner) experiment(hatObj, expectedBallsObj, numberOfDraws, numberOfExperiments) → {Number|String}

A function that perform an N number of draws on the applied Object, to determine the percentage that a given draw occurs. It uses a for loop and a utility function, to perform on an Object. If the number of balls or their color are not in the Hat Object, the function will return an error message.
