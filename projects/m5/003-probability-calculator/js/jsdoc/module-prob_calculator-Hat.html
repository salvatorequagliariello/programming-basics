Methods

draw(numberOfBalls) → {Array.<string>}

Takes a number as argument. The method mimin a draw from an Hat Object. The draw is completely random as it is based on Fisher-Yates algorithm. It shuffle the contents Array of the Object and pick the first entries, to push it to another Array, and delete it from the contents Array. If the Number used as argument is greater than the Array size, the nethod returns the whole array.
